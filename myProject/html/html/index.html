Kinh nghiệm lập trình của tôi

HTML / CSS và Bootstrap

- Sử dụng tương đối thuần thục HTML/CSS và kết hợp thư viện Bootstrap để tạo một trang Web. - Có khả năng phân tích và lập trình trang Web cần thực hiện.
- Hiểu và đang trong quá trình hoàn thiện khả năng Responsive, tương thích trang Web cho các thiết bị di động khác nhau.

JavaScript / jQuery / DOM

- Tạo ứng dụng Website bằng JavaScript: tính năng tương tác như âm thanh, video, trò chơi đơn giản, khả năng cuộn, hoạt ảnh trên trang..v.v...
- Xây dựng các tính năng tương tác: CRUD, tương tác với API, xây dựng thuật toán các trang Admin quản lý trang Web.
- Hiểu được nền tảng của DOM, khả năng dùng jQuery kết hợp với JavaScript để tiết kiệm thời gian xây dựng trang Web.
- Refactoring code dựa trên convention theo yêu cầu để source code dễ đọc, dễ phát hiện lỗi sai, dễ phát triển và dễ bảo trì hơn. - Sử dụng tốt công cụ test và debug của các nền tảng duyệt trình.

Framework

- Hiểu được tầm quan trọng của các Framework và sẵn sàng học tập một Framework mới theo yêu cầu công việc.
